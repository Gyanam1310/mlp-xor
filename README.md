# Solving XOR using Multi-Layer Perceptron (MLP)

This project demonstrates how a **Multi-Layer Perceptron (MLP)** can solve the XOR problem, which is not linearly separable and cannot be handled by a simple perceptron.

## 📌 Overview

* Implemented an MLP using `scikit-learn`
* Trained the model on XOR logic gate data
* Visualized the **decision boundary**
* Compared results with simple perceptron behavior

## 🧠 Key Concepts

* Linear vs Non-Linear separability
* Role of hidden layers in neural networks
* Activation functions (logistic, tanh)
* Decision boundary visualization

## 📊 Results

* Model successfully classifies XOR data
* Achieves **100% accuracy** on training data
* Learns a **non-linear decision boundary**

## 🚫 Why XOR is Important

The XOR problem is **not linearly separable**, meaning a single-layer perceptron fails.
This demonstrates why **multi-layer neural networks (Deep Learning)** are required.
